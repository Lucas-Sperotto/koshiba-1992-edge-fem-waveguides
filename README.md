# koshiba-1992-edge-fem-waveguides
Reprodução didática em C++17 do artigo de Koshiba e Inoue (1992), aplicando o método dos elementos finitos com elementos de aresta triangulares à análise de guias de onda de micro-ondas e ópticos, com foco em autovalores, constante de propagação, modos guiados e eliminação de soluções espúrias.
